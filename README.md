# Niche Calc 2

A collection of specialized calculation tools

## Features

- Aspect Ratio
- Cost Per Year
- Network Address
- Packets to Bytes Conversion
- Pixels Per Inch

## Setting Up the Development Environment

1. Clone the repository
2. Install dependencies: `npm install`
3. Launch the development server: `npm run dev`

## Running Tests

`npm test`

## Building the Project

`npm run build`
